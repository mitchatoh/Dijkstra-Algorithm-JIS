Instructions:

1) Click the dijkstra project.exe file in the Release located in the x64 within the Release folder

C:dijkstra project\x64\Release

2) use arrow keys to position the starting node (Red)
3) click enter when done  
4) use arrow keys to position the first end node (Yellow)
5) click enter when done 
6) repeat 4 and 5 for the second end point
7) Use the mouse to create obstacles (i.e. draw the layout of Taylor 200) 
8) Press space bar to run Dijkstra's algorithm
